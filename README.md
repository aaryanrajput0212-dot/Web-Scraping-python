# Web-Scraping-python
A Python web scraping application that extracts quotes and author names from a website using Requests and BeautifulSoup and displays them in a user-friendly format.

## Features

* Fetches data from a website
* Extracts quotes automatically
* Displays author names
* User-friendly menu system
* Error handling for network issues

## Technologies Used

* Python
* Requests
* BeautifulSoup4

## How It Works

1. Sends a request to the website.
2. Downloads the HTML content.
3. Parses the HTML using BeautifulSoup.
4. Extracts quotes and authors.
5. Displays the information in the terminal.

## Menu

```text
===== WEB SCRAPER =====
1. View Quotes
2. Exit
```
